## MCP

Remato also exposes a [Model Context Protocol](https://modelcontextprotocol.io) server so AI assistants (Claude, Cursor, VS Code Copilot, Windsurf, Zed, Codex CLI, Gemini CLI, …) can call the external API as tools. See [mcp.md](./mcp.md) for the endpoint and per-client installation guides.

Feel free to report any issues or feature requests in the [Issues](https://github.com/rematocorp/external-api/issues).
